 App Launch
   │
   ├─[no profile]──► ONBOARDING (multi-step wizard)
   │                   Step 1  welcome    — intro screen
   │                   Step 2  name       — username input
   │                   Step 3  date       — sober date picker (datetime-local)
   │                   Step 4  spend      — daily spend + currency selector
   │                   Step 5  security   — lock method: None / Biometric / PIN
   │                   Step 6  pin        — 4-digit PIN setup (if PIN chosen)
   │                   Step 7  notifications — toggles (motivations, reminders,
   │                           milestones) + morning/evening time pickers +
   │                           frequency (auto/gentle/light/minimal)
   │                   Step 8  finish     — completion screen
   │                     └──► saves UserProfile → enters App
   │
   ├─[lockMethod ≠ 'none']──► LOCK SCREEN
   │                            Biometric mode:
   │                              Auto-triggers on mount
   │                              Error states: lockout, no enrollment,
   │                                no hardware, cancelled, unavailable
   │                              Fallback to PIN if configured
   │                            PIN mode:
   │                              Masked numeric input (dots)
   │                              4-8 digit verification
   │                            [unlock] ──► App
   │
   └─[profile exists + unlocked]──► MAIN APP SHELL
                                      SafetyModal (crisis check on launch)

 SLIP SCREEN  [screen='slip']  ← back → settings
   6-step compassionate workflow:
   Step 1  Intro       — gentle messaging: "A slip is a data point,
                         not a defeat." + "your data and history all stay."
   Step 2  When        — datetime picker + previous streak display
   Step 3  Trigger     — 12 trigger tags (multi-select):
                          Stress · Loneliness · Boredom · Anger · Exhaustion
                          Social pressure · Celebration · Anxiety · Sadness
                          Habit/autopilot · A specific person · A specific place
                          + freetext field
   Step 4  Feeling     — 10 feeling tags (single-select):
                          Numb · Relieved · Guilty · Disappointed · Indifferent
                          Sad · Angry with myself · Confused · Tired · Empty
                          + custom input
   Step 5  Reflection  — freetext ("message to your future self")
   Step 6  Confirm     — summary review + Submit button
                         [submit] → recordSlip() → resets soberDate,
                                    saves Slip record, clears
                                    firedMilestoneDays / firedSavingsTiers,
                                    → setScreen('settings')

 SLIP LOG  [screen='sliplog']  ← back → settings
   ├─ Read-only history of all Slip records
   ├─ Cards grouped by date
   ├─ Expandable: trigger tags / feeling / reflection / previous streak length
   ├─ Editable reflection note per entry
   └─ Delete with confirmation

 HISTORY  [screen='history']  ← back → settings
   ├─ Filter tabs: All · Cravings · Thoughts · Exercise · Sleep · Journal · Gratitude
   │               (each tab shows count badge)
   ├─ 7-day summary stats row
   ├─ Search bar
   ├─ Entries grouped by date (month dividers)
   ├─ Expandable entry detail per type:
   │    Cravings   — intensity, trigger, duration, overcome status
   │    Thoughts   — intensity label, full text
   │    Exercise   — activity, duration, distance, notes
   │    Sleep      — hours, quality (1-5 stars)
   │    Journal    — truncated preview → full text on expand
   │    Gratitude  — full text
   └─ Delete per entry (confirmation modal)

 CRISIS LINES  [screen='crisis']  ← back → settings
   ├─ Emergency callout (911 / 999 / 000 / 112)
   ├─ Withdrawal warning callout (detailed symptom list)

   │    International/US · UK/Ireland · South Africa · Australia
   │    Canada · New Zealand · Europe
   └─ 15+ crisis line cards per region:
        Name · Number · Description · Hours
        Call button (tel: link) · Copy number · Copy website

 RECOVERY GROUPS  [screen='groups']  ← back → settings

 BACKUP & RESTORE  [screen='backup']  ← back → settings
   ├─ Privacy callout ("Your data never touches a server")
