# Screenshot capture checklist

The site currently uses real Journey Forward screenshots already available from the Flutter app:

- `home-hero.jpg`
- `home-journey.jpg`
- `home-reflection.jpg`
- `home-toolkit.jpg`
- `milestone-share.jpg`

To make the site fully complete, add fresh real-device captures from the current luxury build for:

1. `progress-screen.jpg` — Progress / milestone overview
2. `journal-screen.jpg` — Journal screen
3. `toolkit-screen.jpg` — Toolkit or emergency support screen
4. `recovery-timeline-screen.jpg` — Recovery / quitting timeline screen
5. `cbt-screen.jpg` — CBT thought reframe, if you want to promote that feature publicly
6. `slip-support-screen.jpg` — Slip support / return-after-a-hard-day screen, if you want to promote that feature publicly

Keep screenshots honest:

- Use the actual Flutter app, current theme, and real copy.
- Do not use generic phone mockups or invented UI.
- Capture with the same cream / forest visual system visible.
- Prefer screenshots without debug banners.
